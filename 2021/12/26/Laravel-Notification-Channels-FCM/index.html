laravel new fcm_test

composer require laravel/ui

創建 auth

php artisan ui bootstrap --auth

npm install

npm run dev

新增 user 欄位，儲存要發送裝置 token

php artisan migrate

composer require laravel-notification-channels/fcm:~2.0
